## How to Use Each Agent

### AGENT-001: Channel Auditor
**Purpose:** Deep channel reconnaissance to identify high-value targets

**How to Use:**
1. Click on "CHANNEL AUDITOR" card
2. Chat about channels, ask questions, OR share URLs
3. The agent will respond conversationally

**Example Messages:**
```
"What makes a successful YouTube channel?"
"How can I grow my subscriber count?"
"Analyze this channel: https://youtube.com/@mrbeast"
"Compare these channels: https://youtube.com/@mkbhd and https://youtube.com/@veritasium"
"What should I focus on for a gaming channel?"
```

## Troubleshooting

### Agent not responding?
1. Check API status (top-right)
2. Verify your input format
3. Check browser console (F12) for errors

### Error messages?
- Read the error carefully - it will guide you
- Make sure URLs are valid YouTube links
- Ensure input meets minimum requirements

### Need to start over?
- Close the chat interface
- Click the agent card again
- Fresh chat session starts
